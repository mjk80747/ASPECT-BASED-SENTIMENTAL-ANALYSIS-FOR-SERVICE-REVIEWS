Extension
========

In the base paper, the author mentioned to use different techniques for analysis the datasets for predicting the sentiment with different dataset and models, 
As an extension we applied an ensemble method  combining the predictions of multiple individual models to produce a more robust and accurate final prediction.
However, we can further enhance the performance by exploring other ensemble techniques such as LSTM + GRU and  Voting Classifier  for analysis Dataset, 
With the above As an extension we can build the front end using flask framework for user testing with authentication.
